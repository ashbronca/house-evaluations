# House Evaluations

A personal repo for evaluating Australian residential properties before making an offer.

## Structure

- `.claude/skills/property-valuation-au/` — Claude Code skill that researches and estimates market value for a given address. Covers comparable sales, suburb trends, neighbourhood quality, risk factors, and offer guidance.
- `evaluations/` — Property evaluation reports generated by the skill, one per property researched.

## Usage

Paste a property address into Claude Code and the `property-valuation-au` skill will run a full analysis, saving a Markdown report to the `evaluations/` folder.
